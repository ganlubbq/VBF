Description
===========

*E2* is a symmetric block cipher which was created in 1998 by NTT and submitted to the AES competition. E2 has an input transformation and output transformation that both use modular multiplication, but the round function itself consists only of XORs and S-box lookups. The single 8Ã—8-bit S-box is constructed from the composition of an affine transformation with the discrete exponentiation :math:`x^{127}` over the finite field :math:`GF(2^8)`. NTT adopted many of E2's special characteristics in Camellia, which has essentially replaced E2.
