AES analysis
************

Description
===========

*AES* is a block cipher used for securing sensitive but unclassified material by U.S. Government agencies since December 6, 2001 and has become the de facto encryption standard for commercial transactions in the private sector.

SRD is AES 8x8 S-box and it is generated by determining the multiplicative inverse for a given number in :math:`GF(2^8) = GF(2)[x]/(x^8 + x^4 + x^3 + x + 1)`, Rijndael's finite field. SRDInv is the inverse S-box and it is simply the S-box SRD run in reverse.
